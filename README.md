GPG verification Commands:
1. gpg --import public-key.asc - You can use this command to import the public key for verification. Make sure to link the path based on where the key is stored on your device. 
2. gpg --verify "Education System Framework.pdf.asc" "Education System Framework.pdf" - This can be used to specifically verify the signature on my education system framework. 
3. gpg --verify "Student Power Systems.pdf.asc" "Student Power Systems.pdf" - This can be used to specifically verify the signature on my student power systems documentation.
